# homebrew-tap

Homebrew tap for [dsaad68](https://github.com/dsaad68) tools.

## ripple

Experimental batteries-included agent with MLX and Apple Containers, pure Swift.

```sh
brew install dsaad68/tap/ripple
```

Requires Apple Silicon and macOS 26+ (Tahoe). Project: https://github.com/dsaad68/ripple

The `ripple` formula here is generated automatically by ripple's release workflow; do not edit it by
hand. The source template lives in the ripple project.
